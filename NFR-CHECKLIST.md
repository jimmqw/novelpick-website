# 非功能性需求（NFR）检查清单

## 性能 (Performance)
- [ ] 响应时间：P50/P95/P99 分别是多少？
- [ ] 吞吐量：QPS/TPS 支持多少并发？
- [ ] 资源占用：CPU/内存/磁盘使用上限？
- [ ] 缓存策略：哪些数据需要缓存？TTL多长？

## 安全性 (Security)
- [ ] 认证：用户名密码/OAuth/JWT/SAML？
- [ ] 授权：RBAC/ABAC/权限最小化？
- [ ] 传输加密：TLS版本？
- [ ] 静态数据加密：敏感字段是否加密存储？
- [ ] 输入校验：SQL注入/XSS/CSRF防护？
- [ ] 审计日志：哪些操作需要记录？
- [ ] 数据脱敏：日志/错误信息中是否有敏感信息泄露？

## 可用性 (Availability)
- [ ] SLA目标：99.9% / 99.95% / 99.99%？
- [ ] 故障恢复：RTO（恢复时间目标）/ RPO（数据丢失目标）？
- [ ] 多活：单活/同城双活/异地多活？
- [ ] 降级策略：哪些功能可以降级？降级后体验？

## 可扩展性 (Scalability)
- [ ] 水平扩展：加机器就能提升性能？
- [ ] 垂直扩展：单实例性能上限？
- [ ] 分库分表：数据量预估，何时需要分片？
- [ ] 微服务拆分：哪些模块需要独立部署？

## 可维护性 (Maintainability)
- [ ] 监控：Metrics/Logs/Traces 覆盖哪些关键路径？
- [ ] 告警：哪些指标触发告警？告警级别？
- [ ] 部署频率：支持持续交付吗？
- [ ] 文档：架构文档/接口文档/运行手册？

## 兼容性 (Compatibility)
- [ ] 向前/向后兼容：API版本策略？
- [ ] 客户端兼容：支持哪些浏览器/操作系统/APP版本？
- [ ] 数据迁移：老数据如何迁移？迁移窗口？

## 合规性 (Compliance)
- [ ] 数据主权：数据存储在哪个地区？
- [ ] 行业合规：GDPR/CCPA/PIPL/等保？
- [ ] 敏感数据：个人信息/金融数据/医疗数据的处理要求？

## 隐含约束（Implicit Constraints）
- [ ] 预算限制：开发/运维预算上限？
- [ ] 时间约束：上线deadline？
- [ ] 团队技能：现有技术栈和团队熟悉程度？
- [ ] 第三方依赖：哪些外部系统必须对接？SLA如何保证？
- [ ] 用户规模：当前用户量 vs 3年后预估？
- [ ] 网络环境：弱网环境支持？离线功能？
- [ ] 设备多样性：PC/平板/手机/车载/IoT？
