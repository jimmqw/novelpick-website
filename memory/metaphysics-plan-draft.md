## 六、内容体系建设（简→深）

### 内容分级模型

**Level 1：入门（1000词以内）**
- "什么是XXX"
- "XXX适合我吗"
- "XXX与风水有什么区别"
- 目标读者：完全零基础

**Level 2：基础（1500-2500词）**
- "XXX入门教程"
- "XXX十二宫位详解"
- "XXX与四柱八字的关系"
- 目标读者：初学者，有一点基础

**Level 3：进阶（3000+词）**
- "XXX格局大全"
- "XXX实战案例分析"
- "XXX高级技巧揭秘"
- 目标读者：有基础的爱好者

**Level 4：专业（5000+词）**
- "XXX核心算法解析"
- "XXX古籍原文+白话注解"
- "XXX各派系对比研究"
- 目标读者：专业人士

## 七、建站工具与发布流程

### 方案A：GitHub Pages + Jekyll（推荐）
**优点：**
- 免费
- 稳定
- 与AI写作工作流契合（Markdown）

**缺点：**
- 需要Git知识
- 插件有限

**适用场景：** 100-500页规模

### 方案B：WordPress.org
**优点：**
- 插件生态丰富
- SEO插件强大（Yoast, Rank Math）
- 易于管理

**缺点：**
- 需要维护
- 托管费用

**适用场景：** 500+页，长期运营

### 方案C：Webflow
**优点：**
- 设计自由度极高
- 视觉效果好

**缺点：**
- 费用较高
- 学习曲线

**适用场景：** 注重品牌展示

### 推荐工作流（AI辅助）
1. 选题 → 关键词研究
2. AI生成初稿（Claude/我的能力）
3. 人工审核关键内容
4. 翻译+本地化（如需要）
5. SEO优化（标题/描述/H标签/内链）
6. 发布到GitHub Pages/WordPress

### 所需工具清单
- 域名：Porkbun / Cloudflare
- 托管：GitHub Pages
- 建站：Jekyll
- 内容：Markdown编辑器（Typora/Obsidian）
- 翻译：DeepL / ChatGPT
- SEO：Google Search Console + Screaming Frog
- 分析：Google Analytics 4
- 邮箱：ProtonMail / Zoho Mail
