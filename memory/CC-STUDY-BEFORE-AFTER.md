## 一、记忆系统

| 维度 | 学习前（v1.0.1） | 学习后（v1.0.6） | 落地程度 |
|------|----------------|-----------------|---------|
| 记忆分类 | 无分类，MEMORY.md混写 | 四类分类体系（user/feedback/project/reference） | ✅ |
| 禁止保存规则 | 无 | extractMemories规范：代码/调试/临时任务不写入 | ✅ |
| 记忆检索 | 无检索，全文扫描 | 语义检索（findRelevantMemories启发），按需加载≤5个相关记忆 | ✅ |
| 记忆漂移处理 | 无 | DRIFT CAVEAT：观察到的状态 > 记忆，冲突时信任当前 | ✅ |
| 记忆写入方式 | 直接写入 | 提议promotion机制（需用户确认再写） | ✅ |
| 记忆上限管理 | 无 | 25KB/200行限制 + truncation逻辑 | ✅ |
| 遗忘机制 | 无，只增不减 | 无遗忘机制（未实施） | ⏳ |

**源码贡献**：memoryTypes.ts / memdir.ts / findRelevantMemories.ts / paths.ts
